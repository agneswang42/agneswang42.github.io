## Overview

As team lead, I directed the design and development of a Java-based pixel collision game that combines interactive gameplay with object-oriented software architecture. The project emphasized collaboration, software design, and user experience while demonstrating core principles of game development.

## Project Outcome

The completed game includes player and enemy movement, collision detection, level progression, real-time score tracking, and sound effects within a graphical Java interface. The software architecture was planned using UML diagrams before implementation, improving modularity and maintainability.

The project concluded with a formal demonstration and technical presentation to faculty and classmates.
