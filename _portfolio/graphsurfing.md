## Overview

GraphSurfing is a Java-based graph analysis framework that supports efficient storage, traversal, and analysis of large graphs. The project implements multiple graph representations and classic graph algorithms while emphasizing performance, scalability, and clean object-oriented design.

## Project Outcome

The framework supports both adjacency-list and adjacency-matrix graph representations, allowing users to select the most appropriate data structure for different graph densities. It includes efficient edge operations, lazy neighbor iterators, shortest-path search, and strongly connected component detection.

To evaluate scalability, the implementation was tested on a real-world Wikipedia hyperlink graph containing more than **700,000 vertices** and **3.6 million edges**, demonstrating the practical impact of data structure selection on algorithm performance.
