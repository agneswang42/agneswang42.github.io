## Overview

This project recreated a modified version of the classic Atari PONG arcade game on FPGA hardware using Verilog HDL. The project provided hands-on experience with digital system design, hardware verification, and FPGA implementation.

## Project Outcome

The game was designed, simulated, synthesized, and deployed using Altera Quartus Prime. Digital circuits were validated through timing analysis and hardware testing to ensure reliable gameplay. The final implementation demonstrated real-time paddle movement, ball collision logic, score tracking, and stable hardware performance.

Project results were documented through technical reports and presented during demonstrations.
